# Smart Silence — Android Sound Automation App

## 🚀 Quick Overview
Smart Silence is an Android automation app that automatically switches the phone to silent mode during college hours based on a customizable schedule. It helps reduce classroom disturbances and improves focus.

## 📲 Download APK
You can directly install the latest version:

👉 https://github.com/madhan929/smart-silence/raw/main/app-release.apk

## ✨ Features
• Automatic silent/normal mode switching  
• Customizable class schedule  
• Focus mode timer  
• Skip-today option  
• Weekend detection  
• Minimal and clean UI  

## 🛠 Tech Stack
• Kotlin  
• Android SDK  
• AlarmManager  
• SharedPreferences  
• Material UI
